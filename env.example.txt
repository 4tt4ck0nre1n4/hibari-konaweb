# WordPress Headless CMS API Configuration
#
# ============================================
# ローカル開発環境用の設定方法
# ============================================
# 1. このファイルを .env にコピーしてください
#    cp env.example.txt .env
#
# 2. PUBLIC_API_URL をローカルのWordPress URLに変更してください
#    例: http://hibari-konaweb.local
#
# ============================================
# Netlify本番環境用の設定方法
# ============================================
# Netlifyダッシュボードで以下を設定してください：
# Site Settings → Environment Variables
#
# PUBLIC_API_URL=https://hibari-konaweb.com
# PUBLIC_API_PREFIX=/wp-json/wp/v2/
#
# ⚠️ 重要: 本番環境のWordPress URLは公開アクセス可能である必要があります
# ⚠️ ローカルURL（http://localhost, http://hibari-konaweb.local など）は使用できません
#
# ============================================

# ローカル開発環境の例
PUBLIC_API_URL=http://hibari-konaweb.local
PUBLIC_API_PREFIX=/wp-json/wp/v2/

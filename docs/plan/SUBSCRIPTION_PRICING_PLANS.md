# サブスクリプション型ホームページ制作サービス料金プラン開発仕様書

このドキュメントは、料金シュミレーターのプラン「ホームページ制作」の仕様書です。

## サブスクリプション型ホームページ制作サービス料金プラン

料金シュミレーターのプラン、「ホームページ制作」ボタンを選択すると、
「ホームページ制作」画面に切り替わる。

- タブ切り替えのプログラムを追加（アクセシビリティ対応）
- デフォルトのタブは「コーディング」

「ホームページ制作」タブ

- 見出しはサブスクリプション型プラン
- プランは3つ
- swiperでプランを切り替え

## サブスクリプション型プラン名称（プラン名称は仮の名前）

1. エントリープラン（名刺代わり、低予算、テンプレート）
2. ミドルプラン（一般的なホームページ）
3. エキスパートプラン（オリジナル制作、ヘッドレスCMS開発、shopifyなど）
4. オリジナル
